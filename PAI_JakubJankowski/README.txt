## Projekt zaliczeniowy z laboratorium "Programowanie aplikacji internetowych"

## Tematyka projektu: Strona do zamawiania kursów językowych

## Funkcjonalności:
- tworzenie/logowanie na konto
- zamawianie kursów
- możliwa edycja poszczególnych danych dla użytkowników
- panel administratora do usuwania zamówień/użytkowników

## Wymagania
- XAMPP (mySQL Database, APACHE Web Server)
- PHP 8.0

## Uruchomienie
- Folder projektowy PAI_JakubJankowski należy umieścić w 'XAMPP/htdocs'
- W XAMPP włączyć Apache Web Server i MySQL database
- Zaimportować bazę danych klienci.sql z folderu projektowego w zakładce localhost/phpmyadmin
- Otworzyć projekt z folderu htdocs przy pomocy 
	- Apache NetBeans - Open Project
	lub
	- PhpStorm - Open
- Uruchomić aplikację pod adresem localhost/PAI_JakubJankowski

# Uwagi
- Jeżeli wystąpi problem i adres localhost/PAI_JakubJankowski nie będzie działał proszę spróbować localhost/Projekt_JJ
